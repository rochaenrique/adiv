# ADIV - VIDA al reves
Vuelva a la vida y intenta terminar donde has empezado.
Proyecto para la Dimension Game Jam 2025.

## Estagios:
- De vuelta a la vida
- Abuelito
- Hombre de familia
- Universidad
- Instituto
- Chiquitin
- Bebe
- Bienvenido a la vida
